# GitHub Prior Art Research

Research proven solutions and battle-tested approaches before reinventing the wheel.

## When to Use

- "How do I implement X?"
- "What's the best way to handle Y?"
- "Which library should I use for Z?"
- "What tool is best for..."
- Before proposing architectural solutions
- When evaluating technology choices

## Research Strategy

1. **Find Popular Solutions**: Search for highly-starred repositories
2. **Check Real Usage**: Look at actual implementation examples
3. **Review Issues**: Learn from others' problems and solutions
4. **Compare Alternatives**: Evaluate multiple approaches
5. **Check Maintenance**: Verify projects are actively maintained
