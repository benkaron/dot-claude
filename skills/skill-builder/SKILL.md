# Skill Builder

Build focused, discoverable skills that solve specific problems with clear triggers and progressive disclosure.

## When to Use

- Creating new skills for your workflow
- Improving existing skill discoverability
- Testing skill activation patterns
- Organizing complex knowledge into skills
- Building project-specific skills

## Skill Design Principles

1. **Single Purpose**: Each skill solves one specific problem
2. **Clear Triggers**: 3-5 concrete phrases that activate it
3. **Progressive Disclosure**: SKILL.md for discovery, REFERENCE.md for details
4. **Actionable Content**: Provides immediate value, not just information
